# README #

This is the work done as part of the Fullstack.io 'NG Rails' course.

### What is it? ###

* Rails Backend
* Angular MVC

### How do I get set up? ###

* clone this repo
* ensure proper Ruby version installed (use RVM)
* ensure you are on the right version (cd out and back into folder)
* bundle install
* rake db:create
* rails s
